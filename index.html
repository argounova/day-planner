  • About

    Although a day planner needs little introduction, this particular project incorporates some useful features that would almost certainly be found on larger, more complex applications.
    At the core is client-side storage. A perfect and simple solution to a lightweight app such as this.
    The day planner also utilizes Email.JS through a CDN for a more elegant approach to contact. Moment JS and Bootstrap v5.3 round out the libraries that help to elevate the project.
